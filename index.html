Grid vs Flex

Choosing between Flexbox and Grid shouldn't be seen as a dilemma: they are CSS elements and work well together.

Design using grid-layout


							.gallery {
								display: grid;
								grid-template-columns: repeat(4, 120px);
								grid-template-rows: repeat(4, 120px);
								grid-gap: 10px;
							}

							.image {
								height: 120px;
								border-radius: 10px;
								background-size: cover;
								background-position: center;
								transition: transform 0.5s ease;
							}

							.image:hover {
								cursor: pointer;
								transform: scale(1.1);
								filter: brightness(1.2);
								z-index: 1;
							}

							.image:not(:hover) {
								filter: brightness(0.5);
							}

							.current-image {
								grid-column-start: 2;
								grid-column-end: 4;
								grid-row-start: 2;
								grid-row-end: 4;
								border-radius: 10px;
								background-size: cover;
								background-position: center;
								transition: transform 0.5s ease;
							}
